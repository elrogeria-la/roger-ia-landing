Prompt vs Skill vs Agent

La diferencia real que determina qué tan poderoso es tu uso de la IA. El que solo sabe Prompts obtiene respuestas. El que construye Agents, escala.

Nivel 1

Prompt

Una instrucción. La forma más básica de interactuar con la IA. Rápido de hacer, sin consistencia, sin memoria.

Nivel 2

Skill

Conjunto de instrucciones + contexto + ejemplos definidos de antemano. La IA produce resultados consistentes sin que repitas el contexto.

Nivel 3

Agent

Sistema autónomo con memoria, herramientas y capacidad de tomar decisiones. Trabaja solo mientras tú haces otras cosas.

¿Qué es un Prompt?

Un Prompt es cualquier mensaje que le envías a una herramienta de IA. Pero hay una diferencia enorme entre un prompt malo y uno bueno — y esa diferencia no es magia, es estructura.

La anatomía de un buen prompt

ROL → ¿Quién eres tú en este contexto?
TAREA → ¿Qué quieres que haga exactamente?
CONTEXTO → ¿Qué información necesita para hacerlo bien?
FORMATO → ¿Cómo quieres que te entregue el resultado?

Prompt malo vs bueno

Malo: "Escríbeme un email para mis clientes" → email genérico que suena a plantilla corporativa.

Bueno: "Eres un copywriter especializado en pequeños negocios de restauración en México. Escribe un email para clientes de un restaurante familiar en Guadalajara. Objetivo: anunciar el nuevo menú de mariscos. Tono: cálido, familiar, no vendedor. Máximo 150 palabras. CTA: 'Haz tu reservación'." → Email que parece escrito por alguien que conoce el negocio.

Regla de oro: Más contexto = mejor resultado. La IA no adivina. Funciona con lo que le das.

¿Qué es una Skill?

Una Skill es el conocimiento especializado que le instalas a la IA de antemano. En lugar de repetir el contexto cada vez, lo defines una sola vez y la IA lo aplica automáticamente.

Cómo se construye una Skill

IDENTIDAD → Quién es el usuario y para qué usa la IA
AUDIENCIA → A quién van dirigidos los outputs
VOZ DE MARCA → Tono, palabras permitidas, prohibidas
FORMATO → Estructuras de output predefinidas
EJEMPLOS → 2–3 muestras de resultados excelentes
REGLAS → Lo que nunca debe hacer

Ejemplo real: Mi Guionista

El agente Guionista de Roger.IA tiene una Skill que incluye voz de marca ("conversacional, directo, sin tecnicismos"), estructura obligatoria (Hook → Open Loop → Contenido → CTA), palabras prohibidas (innovador, disruptivo, sinergia) y 3 guiones reales de videos que funcionaron.

Resultado: cada guión suena como yo, sigue la estructura correcta y nunca usa las palabras que manchan la marca. Sin configurar nada cada vez.

¿Qué es un Agent?

Un Agent es una IA que puede tomar decisiones, ejecutar acciones y coordinar múltiples tareas de manera autónoma. No solo responde preguntas — actúa en el mundo real.

La analogía del restaurante

Prompt: Le preguntas al cocinero cómo hacer caldo de pollo. Él explica. Tú lo haces.

Skill: El cocinero conoce todas tus recetas favoritas. Cuando dices "haz algo para 4 personas", ya sabe qué prefieres.

Agent: El cocinero tiene acceso a tu refrigerador, puede ir al súper, tiene el presupuesto del día y la lista de invitados. Cuando llegas, la comida está lista. Tú no hiciste nada.

Lo que hace diferente a un Agent

MEMORIA → Recuerda interacciones pasadas
HERRAMIENTAS → Puede buscar en internet, enviar emails, actualizar bases de datos
RAZONAMIENTO → Descompone problemas complejos en pasos y los ejecuta
COORDINACIÓN → Puede orquestar otros agentes
AUTONOMÍA → Actúa sin esperar instrucción para cada paso

Casos de uso por Industria

🏠 Inmobiliaria

Prompt: "Escribe una descripción de esta casa de 3 recámaras en Mazatlán para MLS."

Skill: Sistema entrenado con portafolio, precios y zonas. Genera descripciones, responde compradores y califica leads.

Agent: Cuando entra un lead, califica el perfil, hace match con propiedades, envía propuesta personalizada y agenda la cita. El agente humano solo va a la visita.

🍽️ Restaurante

Prompt: "Escribe 5 posts para Instagram para promover mi nuevo menú de mariscos."

Skill: Community Manager IA entrenado con el menú, precios, horarios y tono de marca. Genera contenido semanal y responde comentarios.

Agent: Monitorea reservaciones, envía confirmaciones por WhatsApp, recuerda a clientes 2 horas antes y pide el review de Google después. Sin intervención humana.

📱 Creador de Contenido

Prompt: "Escribe un hook para un video sobre los 3 errores más comunes al usar ChatGPT."

Skill: Los 6 agentes de Roger.IA — Guionista, Estratega, Copy-Roger, Guardián de Marca, Growth Agent y Director de Arte.

Agent: Analiza rendimiento semanal, genera el calendario siguiente, escribe guiones base y los somete al Guardián. El creador solo graba y aprueba.

⚠️ La trampa más común

Saltar al Agent antes de dominar el Prompt.

Si tu Prompt es malo, tu Agent escala resultados malos. La base siempre importa.

El orden correcto: Domina el Prompt → Construye Skills → Automatiza con Agents.
